== Description == 

A simple, attractive, extremely fast and light-weight pop-up/lightbox for your WordPress Blog. Supports custom design and multiple content sources.


== Installation ==

1. Upload the 'pluginbear_popup' folder to the '/wp-content/plugins' directory 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to PluginBear > Popup and choose your settings
4. That is all!
